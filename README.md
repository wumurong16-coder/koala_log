# 考拉小狗日记 - 前端界面

## 项目简介

考拉小狗日记的前端界面，使用原生HTML/CSS/JavaScript开发。

## 功能模块

### 1. 健康日志
- 饮食记录：记录食物类型、分量、热量、喂食时间
- 便便健康：记录排便时间、健康评分（1-5级）、颜色、备注
- 遛狗记录：记录开始/结束时间、路线、距离、卡路里消耗
- 统计图表：可视化展示每日/每周的健康数据趋势

### 2. 成长相册
- 时间轴相册：按月分类的瀑布流布局
- 智能标签：AI自动识别场景，支持手动添加标签
- 成长指标：记录体重、身长等数据

### 3. 智能助手
- 健康咨询：基于记录数据提供个性化建议
- 训狗技巧：根据狗狗年龄推荐训练项目
- 预警系统：异常模式检测和提醒
- 对话式UI：聊天界面交互

### 4. 社交网络
- 关系图谱：可视化展示联系人关系
- 互动记录：记录社交活动
- 分享功能：创建分享链接，支持权限控制

## 使用方法

1. 确保后端服务正在运行（默认 `http://localhost:3000`）
2. 直接在浏览器中打开 `index.html` 文件
3. 或使用本地服务器（推荐）：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server
   ```
4. 访问 `http://localhost:8000`

## 主要特性

- 响应式设计，支持移动端和桌面端
- 使用Chart.js进行数据可视化
- 原生JavaScript，无需构建工具
- 模块化代码结构，易于维护

## 注意事项

1. 需要先注册/登录才能使用功能
2. 需要先创建宠物才能添加记录
3. 照片上传需要后端服务支持文件上传
4. AI功能需要配置智谱API密钥

## 浏览器兼容性

- Chrome/Edge (推荐)
- Firefox
- Safari
- 不支持IE浏览器
